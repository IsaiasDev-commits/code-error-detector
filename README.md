# Code Error Detector Extension

This Chrome extension detects common errors in HTML, CSS, and JavaScript code. It will help you improve the quality of your code by identifying syntax and accessibility errors.

## Installation

1. Clone the repository:
   https://github.com/IsaiasDev-commits/code-error-detector.git

2.Open Chrome and go to chrome://extensions/

3.Activate "Developer Mode" in the top right corner

4.Click "Upload Unzipped" and select the project folder.
